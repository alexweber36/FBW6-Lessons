## Variables
Declare a variale
```javascript
let message;

message = 'Hello'; // store the string

or 

let message = 'Hello';
```
multi word variables
```javascript
let firstName= 'John';  //camel case
let first_name= 'Sara';  //underscore
let FirstName= 'Mansour';  //pascal case
let firstname= 'Anna'
```
LET
```javascript
let  name;
name = 'John Doe';
console.log(name);
name = 'Steve Smith';
console.log(name);
```
CONST
```javascript
const  greeting= 'Hello';
console.log(greeting);
Can  not  reassign
greeting= 'Hi';

Have  to  assign  a  value
const  lastName;
```
